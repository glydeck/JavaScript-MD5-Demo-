Using MD5 for a WEB Page Password

How to Use:

  1. Password is md5demo
  2. Enter the password and click on the Submit button.
  3. Do not use the Enter key
  4. To change the password go to md5hashgenerator.com
  5. Type in the new password and click on the generate button
    Step 4 & 5 can also be done with the linux command line
  6. Copy the new MD5 checksum for the new password
  7. Replace the MD5 checksum in p.js line 3 with the new MD5
